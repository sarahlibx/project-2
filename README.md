  ## :pencil: Description

  The TBR Book Tracker is an app where users can create a virtual bookshelf of books they have read, want to read or are currently reading! Users can also rate and the books they have read. Users have the option to manually add a book or search the Google Books API; once on the shelf, users can modify the status of each book. Additionally, users can connect with fellow readers on the app and view their bookshelves for inspriation on their next reads -- or to connect over a book they both read!

  ## :fire: Getting Started

<details open>
  <summary> How to Create an Account </summary>
    1. Click on 'sign up' button.
    2. Create a username and password (confirm that username is not already in use!).
    3. Click on the 'bookshelf' button to get started adding books!
</details>

<details>
  <summary> How to Connect With the Community </summary>
    1. Ensure you are signed in.
    2. Navigate to the 'community' button.
    3. Select a user and view the books on their shelf!
</details>
